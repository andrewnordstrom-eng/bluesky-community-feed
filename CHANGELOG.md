# Changelog

## [1.0.0] — 2026-03-06

### Added
- Community governance with epoch-based weight voting
- Five scoring components: recency, engagement, bridging, source diversity, relevance
- Bluesky sendInteractions API support
- Private feed access gating with participant management
- Admin dashboard with governance controls, interactions, announcements, audit log
- OpenAPI documentation at /docs
- Pre-commit hooks (husky + lint-staged)
- Dependabot dependency management
- Shared API types between frontend and backend (`src/shared/`)
- Code generators for scoring components and routes
- Transparency endpoints for public governance data
- Bot integration for epoch announcements
